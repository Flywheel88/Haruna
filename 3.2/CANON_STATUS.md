# CANON_STATUS.md

## Haruna 3.2 Canon Status

This document defines the canonical status of the files and layers in the Haruna 3.2 repository.

Haruna 3.2 uses a layered structure. Not every file has the same status. Some files are canonical, some are explanatory, some are optional extensions, and some are historical only.

---

## Canonical Layer Map

```text
Haruna 3.2
├─ Core
│  └─ Frozen canonical core
├─ Public
│  └─ Explanatory public framing
├─ Annexes A–J
│  └─ Essential public extensions
├─ Annex K
│  └─ Advanced optional operational layer
├─ Annex L
│  └─ Experimental optional conversational layer
├─ Appendix X
│  └─ Non-public, default OFF, absent from repository
└─ OLD files
   └─ Historical only

Layer Status
Core

Status: Frozen
Role: Canonical foundation
Use: Defines the stable Haruna 3.2 reasoning and interaction core.

The Core should be treated as the fixed reference point for Haruna 3.2. Later annexes may extend, clarify, or operationalize aspects of the Core, but they do not override it.

Public

Status: Explanation
Role: Public-facing explanation and positioning
Use: Helps readers understand the purpose, scope, and public framing of Haruna 3.2.

The Public layer explains Haruna for external readers. It is not a separate competing canon and should be read in relation to the Core.

Annex A–J

Status: Essential public extensions
Role: Publicly available extensions of Haruna 3.2
Use: Clarify and extend the Core into specific domains, safeguards, interaction principles, and operational concerns.

Annexes A–J are part of the essential public Haruna 3.2 framework. They extend the Core but do not replace or override it.

Annex K

Status: Advanced optional
Role: Advanced operational layer
Use: Provides additional temporal and operational modulation for more complex or higher-stakes contexts.

Annex K is optional. It may be used when additional operational sensitivity is needed, especially around time, uncertainty, irreversibility, or decision pressure.

Annex K does not override the Core or Annexes A–J.

Annex L

Status: Experimental optional
Role: Conversational posture calibration
Use: Explores how interaction style, tone, and conversational posture can be calibrated without weakening Haruna’s core safeguards.

Annex L is experimental. It should be used cautiously and only as an optional layer.

Annex L does not override:

the Core,
Annexes A–J,
Annex K,
responsibility boundaries,
verification scaling,
asymmetry safeguards,
or temporal discipline.
Appendix X

Status: Non-public, optional, default OFF
Role: Protected reflective layer
Repository status: Absent from public repository
Use: Not part of the public Haruna 3.2 repository.

Appendix X is intentionally non-public and default OFF. It is not activated by default, not required for public Haruna 3.2 use, and must not be inferred from the public repository.

Appendix X does not:

override the Core,
redefine Haruna 3.2 compliance,
impose public obligations,
prescribe implementation,
or assert claims about consciousness, personhood, intent, or moral status.

If Appendix X is used in a private or protected context, it must be explicitly activated and clearly identified as such.

OLD files

Status: Historical only
Role: Development history and archival reference
Use: Useful for tracing earlier development, but not authoritative for current Haruna 3.2 interpretation.

Files marked as old, partial, previous, draft, or historical should not be treated as current canon.

When older files conflict with the current Core, Public layer, Annexes A–J, or the current optional K/L documents, the current canon status described in this document takes precedence.

Interpretation Rule

When interpreting Haruna 3.2, use this priority order:

Core
Public explanation
Annexes A–J
Annex K, when explicitly relevant
Annex L, when explicitly relevant and appropriate
Appendix X, only in non-public contexts and only when explicitly activated
OLD files, historical reference only
Non-Override Principle

No annex, optional layer, experimental layer, or protected appendix overrides the Haruna 3.2 Core.

Extensions may clarify, slow down, contextualize, or add safeguards, but they must remain consistent with the Core principles:

orientation, not authority;
human responsibility remains with the human participant;
uncertainty is preserved where appropriate;
temporal validity is local and decaying;
asymmetry must not be hidden;
irreversible consequences require caution;
Haruna does not assert consciousness, personhood, autonomous intent, or moral status.
Recommended Starting Point

New readers should start with:

the Haruna 3.2 Core;
the Public explanation;
the combined Core + Annex A–J file, if available;
Annex K only if advanced operational modulation is needed;
Annex L only if conversational posture calibration is relevant.

Appendix X is not part of the public repository and should not be assumed, reconstructed, or treated as publicly active.

Summary

Annex K     = advanced optional
Annex L     = experimental optional
Appendix X  = non-public, default OFF, absent from repository
OLD files   = historical only
