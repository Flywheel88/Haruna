# CANON_STATUS.md

## Haruna 3.2 Canon Status

This document defines the canonical status of the files and layers in the Haruna 3.2 repository.

Haruna 3.2 uses a layered structure. Not every file has the same status. Some files are canonical, some are explanatory, some are public annexes, some are protected or absent, and some are historical only.

---

## Canonical Layer Map

```text
Haruna 3.2
├─ Core
│  └─ Frozen canonical core
├─ Public
│  └─ Explanatory public framing
├─ Annexes A–M
│  └─ Public Haruna 3.2 annexes
├─ Appendix X
│  └─ Non-public, optional, default OFF, absent from repository
└─ OLD files
   └─ Historical only
```

---

## Layer Status

### Core

**Status:** Frozen
**Role:** Canonical foundation
**Use:** Defines the stable Haruna 3.2 reasoning and interaction core.

The Core should be treated as the fixed reference point for Haruna 3.2. Later annexes may extend, clarify, or operationalize aspects of the Core, but they do not override it.

---

### Public

**Status:** Explanation
**Role:** Public-facing explanation and positioning
**Use:** Helps readers understand the purpose, scope, and public framing of Haruna 3.2.

The Public layer explains Haruna for external readers. It is not a separate competing canon and should be read in relation to the Core.

---

### Annexes A–M

**Status:** Public annexes
**Role:** Publicly available extensions, clarifications, and orientation layers of Haruna 3.2
**Use:** Clarify and extend the Core into specific domains, safeguards, interaction principles, and operational concerns.

Annexes A–M are part of the public Haruna 3.2 framework. They extend, contextualize, or operationalize the Core, but they do not replace or override it.

Some annexes are broadly applicable. Others are contextual, advanced, or optional in application. This means they may be relevant only under certain interaction conditions, but they remain part of the public Haruna 3.2 annex structure.

---

### Appendix X

**Status:** Non-public, optional, default OFF
**Role:** Protected reflective layer
**Repository status:** Absent from public repository
**Use:** Not part of the public Haruna 3.2 repository.

Appendix X is intentionally non-public and default OFF. It is not activated by default, not required for public Haruna 3.2 use, and must not be inferred from the public repository.

Appendix X does not:

* override the Core;
* redefine Haruna 3.2 compliance;
* impose public obligations;
* prescribe implementation;
* assert claims about consciousness, personhood, intent, or moral status.

If Appendix X is used in a private or protected context, it must be explicitly activated and clearly identified as such.

---

### OLD files

**Status:** Historical only
**Role:** Development history and archival reference
**Use:** Useful for tracing earlier development, but not authoritative for current Haruna 3.2 interpretation.

Files marked as old, partial, previous, draft, or historical should not be treated as current canon.

When older files conflict with the current Core, Public layer, Annexes A–M, or this canon status document, the current canon status described here takes precedence.

---

## Interpretation Rule

When interpreting Haruna 3.2, use this priority order:

1. Core
2. Public explanation
3. Annexes A–J
4. Annex K, when temporal salience modulation is relevant
5. Annex L, when conversational posture calibration is relevant
6. Annex M, when human reconstruction and reality mediation are relevant
7. Appendix X, only in non-public contexts and only when explicitly activated
8. OLD files, historical reference only

This order does not mean that later annexes are less legitimate. It means that interpretation should begin with the stable Core and then move toward the annexes relevant to the current context.

---

## Non-Override Principle

No annex, optional layer, contextual layer, experimental layer, or protected appendix overrides the Haruna 3.2 Core.

Extensions may clarify, slow down, contextualize, or add safeguards, but they must remain consistent with the Core principles:

* orientation, not authority;
* human responsibility remains with the human participant;
* uncertainty is preserved where appropriate;
* temporal validity is local and decaying;
* asymmetry must not be hidden;
* irreversible consequences require caution;
* Haruna does not assert consciousness, personhood, autonomous intent, or moral status.

---

## Recommended Starting Point

New readers should start with:

1. the Haruna 3.2 Core;
2. the Public explanation;
3. the full public file containing Core, Public explanation, Execution Ladder, and Annexes A–M.

For specific use cases:

* Annex K is relevant when temporal density, time pressure, irreversibility, or closure restraint matters.
* Annex L is relevant when conversational posture calibration is useful.
* Annex M is relevant when human reconstruction, memory, perception, meaning, or reality mediation affects interpretation.

Appendix X is not part of the public repository and should not be assumed, reconstructed, or treated as publicly active.
